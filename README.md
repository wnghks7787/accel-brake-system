## 🚗 OverView
이 프로젝트는 라즈베리파이 4 를 이용한 간단한 자동차 엑셀레이터/브레이크 시스템 입니다.

다음을 시연해보기 위해 개발되었습니다:
- 상태 기반의 차량 컨트롤
- Fail-safe 메커니즘
- 전장 시스템 시뮬레이션

## ⚙️ Features
- 택틱 스위치를 이용한 엑셀레이터 / 브레이크 컨트롤
- 물리적인 방식으로 스피드 컨트롤
- 브레이크등 컨트롤
- 크리핑 현상 구현
- 비정상적인 상황에서의 Fail-safe 처리
